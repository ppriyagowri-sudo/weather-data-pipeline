# 🌦️ Weather Data Pipeline using Python & SQL

## 📌 Project Overview
This project demonstrates a real-time data pipeline using Python and OpenWeather API. It extracts, processes, and stores weather data into a database.

## ⚙️ Technologies Used
- Python
- Requests (API)
- SQLite
- Git & GitHub

## 🚀 Features
- Fetches real-time weather data using API
- Processes JSON data into structured format
- Stores data in SQLite database
- Automated data pipeline workflow

## 📊 Output Example
- City: Bengaluru
- Temperature: 29.75°C
- Humidity: 48%
- Weather: scattered clouds

## 💡 Use Case
This project simulates real-world data engineering workflows involving API integration, data transformation, and storage.
